# WanderLens Photography Services

A photography portfolio and services website built with React. Showcasing professional photography collections, events, and blog content with an elegant user experience.

## Features

- Multilingual Support - Full English and French translations (Translation files are located in `src/translations/`.)
- Dynamic Collections - Beautiful image galleries with search functionality
- Events Showcase - Display photography events and bookings
- Blog Integration - Share photography stories and insights
- Responsive Design - Optimized for all devices
- Modern UI - Clean, professional interface with modular CSS

## Tech Stack

- React - Frontend framework
- React Router - Navigation and routing
- CSS Modules - Scoped styling

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/taradenaud/WanderLens-Phtotgraphy-Services.git
cd WanderLens-Phtotgraphy-Services
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## Pages

- **Home** - Landing page with featured content
- **Collections** - Photography portfolio galleries
- **Events** - Upcoming and past photography events
- **Blog** - Photography articles and updates
- **About** - Information about the photographer
